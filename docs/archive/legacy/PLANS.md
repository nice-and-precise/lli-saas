# Phase 1: Foundation Scaffold

## Goal

Stand up a fresh `lli-saas` monorepo with runnable service stubs, deployable container definitions, baseline CI/CD, and pilot-oriented docs.

## Plans

1. `01-01` Bootstrap repo, GSD planning state, and root documentation.
2. `01-02` Build service scaffolds for `lead-engine`, `crm-adapter`, and `user-portal`.
3. `01-03` Add infrastructure, CI/CD workflows, and operating docs.

## Done When

- GSD-compatible `.planning` state exists and Phase 1 plan artifacts are present.
- All three runnable services build and pass their baseline tests.
- GHCR-ready GitHub Actions workflows exist for all services.
- Kubernetes manifests and Helm chart render for all services.
- Developer onboarding and David Whitaker pilot docs are in place.
